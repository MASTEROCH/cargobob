# 🟠 КаргоБоб — мост последних 500 метров

> Telegram Mini App, который соединяет водителей, застрявших в очереди, с пешими курьерами рядом.

### ▶️ Живой прототип: **https://masteroch.github.io/cargobob/**

*(откроется как телефон — кликабельно всё, можно переключаться между ролями)*

---

## В чём боль

Топливный кризис. Многокилометровые очереди на АЗС. Водитель стоит часами в жаре, выйти нельзя — потеряешь место. Магазин в 500 метрах недосягаем. Эту дистанцию сегодня не закрывает никто.

## Решение

Независимый **раннер** (пеший курьер) собирает витрину из товаров пешей доступности и доносит к машине за минуты. Платформа — нейтральная доска между двумя частными лицами:

- 💸 **Оплата товара — P2P** напрямую (нал/СБП). Платформа денег за товар не касается, 0% с товара.
- ⭐ **Монетизация — только Telegram Stars** за цифровой доступ (пакеты доставок).
- 📍 **Адресация** — гео-пин + фото машины + описание ряда.
- ✅ **Закрытие — по QR**. Двусторонние рейтинги, отзывы под псевдонимом.

## Как попробовать

1. Открой [прототип](https://masteroch.github.io/cargobob/).
2. Выбери роль **«Я в машине»** (водитель) или **«Я раннер»** — переключается в профиле в любой момент.
3. **Водитель:** радар раннеров рядом → витрина → заказ → чат → закрытие по QR.
4. **Раннер:** включи витрину → собери товары (каталог в один тап) → лови заказы в ленте → закрой по QR → Stars-подписка.
5. 🌍 Язык RU/EN переключается в шапке.

## Стек

Single-file Telegram Mini App (vanilla JS) · Telegram WebApp SDK · Liquid Glass UI (iOS 26/27) · фирменный маскот Боб · RU/EN i18n. Боевой стек переиспользует обкатанную инфраструктуру — MVP за ~3 недели.
